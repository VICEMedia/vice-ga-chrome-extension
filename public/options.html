Google Analytics and Segment Helper Configuration

How to update the labels for Custom Dimensions, Metrics, and Content Groups

By default labels are just the literal dimension index (eg Custom Dimension 1). By updating your options preference you can update the label to reflect the true representation of that label.

Sample Configuration File

Here is an example of how an Configuration File should be structured

  
    {
      "UA-XXXXXX-YY":{
        "contentGroups" :{
          "cg1":"Article UID",
          "cg2":"Topics"
        },
        "customDimensions" :{
          "cd1":"Article UID",
          "cd2":"Topics"
        },
        "customMetrics" :{
          "cm1":"Video Plays - Manual",
          "cm2":"Video Plays - AutoPlay"
        }
      }
    }
